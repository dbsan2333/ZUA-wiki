# 宿舍<Badge type="warning" text="当前页面待完善" />

龙子湖校区目前共有26栋宿舍楼（一般是2或3栋楼连城一体），其中1~6号楼在中心广场周围，建成较早，但内部翻新过。其他宿舍楼后建成，部分已翻新。除24~25号楼为民航学院宿舍楼（但也混有其他学院学生）外，一般每个年级集中在一起，其中按学院、专业集中，剩余少部分专业分散到空余宿舍。

## 宿舍环境

龙子湖校区宿舍主要为6人间，其次有少量4人间。宿舍楼配有卫生间、洗澡间、洗衣机、洗鞋机、直饮水机等，宿舍内有暖气和空调。

六人间均为4上下铺+2上铺+2桌子。每层楼有公共卫生间和洗澡间。
