# 日常消费

在校园中，主要的消费场景为在食堂、商业街、商店的消费和洗澡热水、饮用水消费。

其中，食堂和校医院等场所只允许使用统一的卡机，因此需要使用一卡通或完美校园APP（可绑定支付宝或微信）支付。饮用水机可以使用支付宝扫码或者刷水卡使用。

## 一卡通

[一卡通](https://xxglzx.zua.edu.cn/info/1220/1645.htm)与[完美校园APP](https://app.59wanmei.com/new/index.html)同源，是每个人校内唯一的消费账户。

除了消费场景，一卡通载有学生信息，还作为学生的凭证，可以刷~~学校大门、宿舍~~、图书馆等场所的门禁。（学校大门和宿舍的门禁现已改为人脸识别，图书馆门禁仍需使用一卡通）。期末考试时一卡通可代替学生证（以考试时具体要求为准）。

![alt text](/img/docs/life/consumption/本科生一卡通.png)

一卡通与完美校园的绑定、使用、充值、挂失等操作，详见：

- [信息管理中心：校园一卡通使用说明](https://xxglzx.zua.edu.cn/info/1220/2673.htm)
- [信息管理中心：校园一卡通常见问题解答](https://xxglzx.zua.edu.cn/info/1220/3202.htm)

一卡通由学校信息管理中心负责，如有问题可以在工作日工作时间前往`02A104`[信息管理中心](https://xxglzx.zua.edu.cn/)咨询。

::: info 💡 NFC拷贝
目前大部分手机都支持NFC拷贝功能。由于一卡通是加密卡，NFC拷贝仅能复制卡号，因此在消费场景下无法使用。但对于门禁场景，如图书馆门禁可以正常使用。
:::

## 饮用水

学校宿舍、教学楼、实验楼、图书馆、办公楼等场所都安装有饮用水机，提供净化过的直饮冷水和热水。

使用支付宝扫码饮水机上的二维码或者NFC碰一碰，弹出小程序即可使用。也可以去后勤服务大厅免费办理实体水卡，刷卡使用。

但这里更推荐直接绑定一卡通或NFC，一是减少卡片数量，~~二是扫码小程序会有烦人的广告~~。

::: details 💡 一卡通/NFC绑定饮水机教程
饮水机饮水使用的支付宝小程序的账号扣款，但该小程序可以绑定“云卡”，本质是将卡片的ID（并非学号）与你的支付宝小程序账号相绑定。当放上卡片时，机器读取ID，系统将自动从对应的账户扣款（前提是支付宝对该小程序授权了免密支付）。

**操作步骤：**

将一卡通（或其他卡片）放置在饮水机上的IC卡区域，机器上将显示一串数字并提示“无效卡”，该串数字即为`卡片ID`。

![将卡片放在饮水机上](/img/docs/life/consumption/将卡片放在饮水机上.jpg)

登录支付宝小程序，进入“我的”-“云卡”-“添加云卡”，选择“添加”-“添加云卡”，输入`卡片ID`，并完成添加。

![小程序选择云卡](/img/docs/life/consumption/小程序选择云卡.jpg)

绑定成功后即可直接使用该卡刷卡用水。同时**建议设置每日限额，以防卡片丢失后被盗刷损失过多**。

![绑定云卡](/img/docs/life/consumption/绑定云卡.jpg)

如果想要使用手机NFC刷卡饮水，手机复制该卡片即可。
:::

## 食堂

龙子湖校区有`东苑`和`西苑`2个食堂，其中东苑2楼有`民族餐厅`，提供清真饭菜。另外东苑、西苑三楼均匀包间或教师餐厅。不同的餐厅由不同的公司承包。

食堂窗口消费统一使用卡机，只能刷一卡通或完美校园或绑定完美校园的支付宝/微信扫二维码。窗口价格不贵，品种多样，可自行选择，一日三餐日均消费20~30左右。

当在食堂中遇到问题，可以当场向餐厅经理寻求帮助，或向学校[伙管会](https://mp.weixin.qq.com/s/T6YyfIU_E7Vah4a_QR_G6A)反映。

## 其他商业体

除食堂外，学校还有“商业街”、超市、药店、水果店、干洗店、打印店等，宿舍楼附近也有小商店。这些不限制使用卡机，可自由支付。

### 商业街

位于学校南门口。2024年初开业，由原洗浴中心改造而来，属于综合商业体，共2层。

一楼主要为餐饮小吃，有肯德基、瑞幸、蜜雪冰城等众多品牌入住。二楼除餐饮外，还有台球厅、打印店、考研机构、理发店、“郑航映像”文创店等。

### 超市

学校南门有校内最大的超市。部分宿舍楼附近也有小商店。

### 理发店

校内有3家理发店：西苑餐厅东边2号宿舍楼下、东苑餐厅北边8号宿舍楼下、商业街2楼。
