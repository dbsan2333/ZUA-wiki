## 大致流程

1. 查询到录取状态。
2. 收到录取通知书
3. **认真查看**录取通知书内各通知和文件，并完成要求的内容。
4. 积极关注学院、学院通知（如微信公众号、**官方**通知群等）。
5. 2024年**8月27日-28日**新生(研究生、本科生、专升本学生)报到注册。
6. 8月29日专升本新生开始上课，9月2日研究生新生开始上课，9月18日本科新生开始上课。

::: danger ⚠ 前排提醒

* 开学季是骗子频发期，无论线上还是线下，请警惕各种形势的推销、拉群等行为。不要随意加入各种所谓的“新生群”、“兼职群”、“勤工俭学群”、“表白墙”等，其中很多是校外人员（处于营销或诈骗目的）所为，请务必注意甄别。

* 姓名、手机号、身份证号码、考生号、银行卡号等为个人敏感信息，请不要随意告诉他人。
* 学费、贷款等涉及资金交易的事情请再三确认，学费交纳请按照通知书里的流程进行，不要向任何个人转账！

:::

## 报到需要准备的材料

1. **录取通知书内要求的材料务必准备齐全**（如录取通知书、高考准考证、身份证、封条完好的学籍档案等）。
2. **根据通知书内说明结合自己需求准备对应材料**，如贷款、户口迁移等。
3. 如果是团员，一般在开学后会进行团组织关系转接，需要团员档案袋、团员证、入团志愿书、介绍信等。
4. 建议准备几份身份证、户口本的复印件备用。
5. 身份证已经或将要过期的同学请及时办理更换。
6. 建议在家提前准备一些一寸、二寸的蓝、红、白底证件照，大学期间在许多场合都需要。同时需要电子版，一般开学后教务系统会要求上传照片。

## 可能需要准备的用品

::: tip 温馨提示

* 生活用品可以从家里带，也可以在学校超市或网上购买，请结合情况决定。

* 每个人情况不一样，下面只提供相对基本的内容。
* 有些不是开学立马需要使用的生活物品（如鞋刷、2B涂卡笔等）可以开学后陆续添置，减轻开学负担。

:::

### 生活用品

1. 清洁用品：毛巾、盆子、牙刷、牙膏、洗面奶、沐浴露、洗衣液、鞋刷等

2. 床上用品：褥子、床单、被子、枕头等（宿舍床的尺寸都是1.9m×0.9m）
3. 学习用品：中性笔、铅笔、2B涂卡、橡皮、笔记本等
4. 其他：卫生纸、雨伞、拖鞋、指甲剪等

### 电子产品

1. 电脑（建议笔记本，台式机需要注意功率；请留意学校通知，部分学院或专业大一上学期可能会禁止带电脑）
2. 排插（为了宿舍安全，请购买大品牌产品）
3. 可充电的小台灯

### 其他

1. 个人必须的和常用的药品

2. 防晒霜（军训用）

3. 秋冬季的衣物（郑州夏季炎热，需要短袖、短裤、短裙等；冬季室内有暖气，但室外温度较低，需要棉袄、羽绒服等）
