# 转专业<Badge type="warning" text="当前页面待完善" />

学校于每年4月份左右（对于新生来说就是大一下学期）开展转专业相关工作，主要对象的大一年级学生，其他年级可能会有额外的限制。

学校总体上施行“转出不限制、转入有要求”政策。转出专业、转出名额、转出条件不设限制（南乌拉尔学院学生只能在本学院各专业之间转入或转出，且不接收其他学院学生转入），转入专业、转入名额、转入条件及考核办法由转入专业所在学院制定并实施。每年转专业信息都会发布在各学院官网上。

除热门专业外，大部分专业相对好转入。具体转专业安排可以参考[《关于做好2025年学生转专业工作的通知》](https://jwc.zua.edu.cn/info/1180/5904.htm)和欲转入专业所在学院的官网通知。

转专业成功后，仍然先完成本学期课程和考试，从当年秋季（对大一的来说就是大二上学期）开始和新专业的班级一起上课。
